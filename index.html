NYC Tree Data Visualization







    About:

    This is the NYC Tree Data Visualization. The focus of this page is to see New York City trees in graphical form by borough. The way to use this application is to select a tree you would like to see and then you can click the button to chart it out.

    The data you are seeing is coming from NYC Open Data 2015 Tree Census. New York's open data platform allows the use of important information of all kinds pertaining to New York City and it's people.
